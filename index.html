XOOPS é uma plataforma para aplicações web escrita em PHP com banco de dados MySQL. XOOPS é orientado a objetos o que torna a ferramenta ideal para o desenvolvimento de sites de comunidades pequenas ou grande, intranets, portais corporativos, blogs e muito mais!

Os sites do oficiais do projeto são:

Projeto no Source Forge: http://sourceforge.net/projects/xoops

Projeto e Comunidadet: http://www.xoops.org

Terceiros apoio ao desenvolvimento: http://dev.xoops.org

XOOPS é distribuído sob os termos da GNU General Public License (GPL) o que significa que é possível usar e modifica. É livre para redistribuir contanto que você concorde com os termos de distribuição do GPL.

O que significa XOOPS?

XOOPS é um acrônimo de eXtensible Object Oriented Portal System. Embora tenha começado como um sistema de portal, hoje ele caminha firmemente como um CMS (Content Management System). XOOPS pode servir como um framework web para uso em sites de pequeno, médio e grande porte.

Pronuncia:

A pronúncia padrão do XOOPS deveria seguir regras inglesas, e você deve pronunciá-lo como “zoo'ps”.

Principais características do XOOPS:

Bases de Dados: XOOPS usa bancos de dados relacionais (atualmente MySQL) para armazenar os dados necessários para a execução de um poderoso sitema de gerenciamento de conteúdo WEB;

Totalmente Modularizado: Os módulos podem ser instalados, desinstalados, ativados e desativados com apenas um clique usando o sistema de administração de módulos do XOOPS. Há milhares de módulos disponíveis para download;

Personalização: Os usuários registados podem editar os seus perfis, selecionar temas do site, fazer upload de avatars personalizados e muito mais;

Gerenciamento de Usuários: Possibilidade de pesquisar usuários por meio de vários critérios, data de registro, data do ultimo login, email, login e etc. É possível enviar mensagens privadas utilizando o sistema de MP ou enviar e-mails para os usuários selecionados;

Apoiado Mundial: O Projeto XOOPS possui muitos sites oficiais de suporte ao redor do mundo, não é necessário falar inglês para solicitar ajuda;

Sofisticado sistema de permissões baseados em grupos: Poderoso sistema de permissões de acesso baseados em grupo, facilmente escolha as seções que determinado grupo de usuários podem ou não acessar no seu site;

Interface Personalizável: XOOPS possui um poderoso sistema de temas. Administradores e usuários podem mudar o visual do site inteiro com apenas um clique. Há também uma infinidade de temas disponíveis para download;

Porque usar XOOPS?

Razões para utilizar o XOOPS como solução para seu projeto WEB:

  1. XOOPS é o primeiro projeto CMS do www.sourceforge.net;

  2. XOOPS já conquistou vários prêmios, e constantemente é finalista como melhor solução CMS em várias competições Open Source ao redor do mundo;

  3. XOOPS é escrito em PHP e usa MySql como banco de dados;

  4. XOOPS usa folhas de estilo (CSS) que permitem mudanças de estilo rápido e uniforme em todo o site;

  5. XOOPS usa um poderoso sistema de templates que pode ser facilmente personalizado para qualquer tipo de necessidade;

  6. XOOPS possui o mais sofisticado sistema de acessos baseado em permissões de grupos, isso permite que usuários de diferentes grupos acessem seções específicas;

  7. XOOPS possui milhares de módulos para gerenciamento de conteúdo de acordo com as necessidades do seu projeto;

  8. XOOPS é Open Source, isso significa que é totalmente gratuito, sem timeouts ou anúncios em seu site;

  9. XOOPS possui uma grande e receptiva comunidade com milhares de desenvolvedores e usuários em todo o mundo.

  10. XOOPS é atualizado e melhorado constantemente para incorporar todos os conceitos mais recentes de websites e padrões WC3;

  11. XOOPS permite o desenvolvimento de temas de uma maneira ágil e simples pois usa Smarty PHP para compilar seus templates para melhor atender às suas necessidades;

  12. XOOPS usa métodos de programação PHP seguros e reconhecidos para garantir a integridade dos dados e segurança do sistema.

  13. XOOPS é um dinâmico Sistema de Gestão de Conteúdo (CMS), isso significa que você pode editar, apagar, acrescentar e atualizar conteúdo on-line instantaneamente.

  14. XOOPS possui milhares de módulos, tais como fóruns, galerias de imagens, calendários, gerenciamento artigos e etc;

  15. XOOPS possui um avançado sistema de notificação por e-mail que permite que os membros do seu site sejam notificados de qualquer atualização no seu conteúdo;

  16. XOOPS possui um painel de administração muito amigável e intuitivo disponível apenas para os administradores do site acessado facilmente pelo browser;

  17. XOOPS não exige nenhum conhecimento de programação, embora isso possa ser útil para a personalização;

  18. XOOPS é reconhecido pelo CMS Watch como um sistema robusto, completo, maduro e bem suportado por um grande público;

  19. XOOPS Smarty PHP em seus temas, o que é uma grande vantagem para os desenvolvedores web e programadores sérios;

  20. XOOPS pode ser facilmente copiados para manter todos os seus arquivos e dados seguros em caso de uma falha do servidor;

  21. XOOPS é multi-lingual, a grande maioria de seus módulos estão disponíveis traduzidos nos principais idiomas;
